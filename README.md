# ZigSys

## 概述
智能家居系统 Android 客户端

## 功能概览
-	用户登录

-	用户注册

-	网关绑定与解绑

-	设备添加与删除

-	设备管理

-	模式调节

-	获取设备状态

-	发送设备命令

## 重要组件

- TabLayout ViewPager 与 Fragment 配合使用实现滑动标签页

- SwipeRefreshLayout 实现下拉刷新

- CardView 卡片单元 更美观 符合MD

- RecyclerView 可理解为加强版ListView
